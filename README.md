# `dilumaluthge/tap` Homebrew Tap

Add the tap with `brew tap dilumaluthge/tap`

## Available formulae

- [Pijul](https://pijul.org/) ([formula](./Formula/pijul.rb)): `brew install dilumaluthge/tap/pijul`
- [Platypus](https://sveinbjorn.org/platypus) ([formula](./Formula/platypus.rb)): `brew install dilumaluthge/tap/platypus`
- [shyaml-rs](https://github.com/0k/shyaml-rs) ([formula](./Formula/shyaml-rs.rb)): `brew install dilumaluthge/tap/shyaml-rs`

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
