# `dilumaluthge/tap` Homebrew Tap

Add the tap with:

```shell
brew tap dilumaluthge/tap
```

## Available formulas

- [Pijul](https://pijul.org/) ([formula](./Formula/pijul.rb)): `brew install dilumaluthge/tap/pijul`
    - A formula for Pijul previously existed upstream, but it [was removed on May 13, 2020](https://github.com/Homebrew/homebrew-core/commit/21702ef2c02ae7a5d925de7aed6defd0beefa93d).
- [Platypus](https://sveinbjorn.org/platypus) ([formula](./Formula/platypus.rb)): `brew install dilumaluthge/tap/platypus`
    - A formula for Platypus previously existed upstream, but it [was removed on October 29, 2024](https://github.com/Homebrew/homebrew-core/commit/d72fd20fcf630707a97b23316c2789d1b46fecb2).
    - A deprecated cask for Platypus [exists upstream](https://formulae.brew.sh/cask/platypus). It is scheduled to be disabled on September 1, 2026.
- [shyaml-rs](https://github.com/0k/shyaml-rs) ([formula](./Formula/shyaml-rs.rb)): `brew install dilumaluthge/tap/shyaml-rs`
    - As far as I can tell, a formula for `shyaml-rs` has never existed upstream.
    - A deprecated formula for the old `shyaml` (the Python-based one) [exists upstream](https://formulae.brew.sh/formula/shyaml). It was deprecated on October 26, 2025.
